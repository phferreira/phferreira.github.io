# phferreira.github.io

A personal portfolio website built with **Flutter** and deployed with **GitHub Pages**.

## About the project

This repository contains the source code for my personal website, where I present:

- A short professional introduction
- Work experience and projects
- Technical skills and stack
- Contact and social links

The application is structured as a Flutter project and optimized for web delivery.

## Tech stack

- Flutter
- Dart
- GitHub Pages

## Running locally

To run the project locally, make sure Flutter is installed and then execute:

```bash
flutter pub get
flutter run -d chrome
```

## Build for web

```bash
flutter build web
```
